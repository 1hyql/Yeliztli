# Variant detail

--8<-- "health-disclaimer.md"

Click any variant in the [Variant Explorer](variant-explorer.md) to open a side panel with a
summary, then **Open full detail** for the full page, organised into tabs:

1. **Overview** — key annotations, ClinVar interpretation, and predicted consequence.
2. **Frequencies** — population allele frequencies across gnomAD subpopulations.
3. **Scores** — in-silico pathogenicity predictions (REVEL, CADD, SIFT, PolyPhen, …).
4. **Literature** — PubMed citations (fetched via NCBI Entrez and cached locally).
5. **Protein** — a Nightingale protein-domain diagram showing the variant's position.
6. **Gene** — gene information and associated phenotypes (MONDO/HPO, plus OMIM if you've added
   a key).

You can generate a single-variant **evidence card** (PDF or PNG) from this page, and jump
straight to the variant in the **[Genome Browser](genome-browser.md)**.
