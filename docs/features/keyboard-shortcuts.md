# Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open the command palette |
| `Escape` | Close a panel or dialog |

## Command palette

Press `Cmd + K` (macOS) or `Ctrl + K` (Linux/Windows) to open the command palette and quickly
search for:

- **Variants** by rsID or gene name,
- **Pages** by name (Dashboard, Settings, any module),
- **Actions** such as switching samples or toggling dark mode.

The command palette is **navigation-only** — it performs no destructive or state-changing
actions.
